# 📚 DeepSeek对话管理器

> 基于berry22jelly的"DeepSeek对话收藏器"改造增强，专为DeepSeek聊天设计的对话管理工具

[![Tampermonkey](https://img.shields.io/badge/Tampermonkey-必备-black)](https://www.tampermonkey.net/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ 功能特点

### 🗂️ **分类管理**
- 固定6个分类（化学/英语/数学/物理/计算机/学校琐事）
- 支持分类重命名（鼠标悬停点击✎即可修改）
- 分类数量实时统计
- 默认"未分类"文件夹收纳未标记对话

### 🔍 **搜索与排序**
- **全文搜索**：支持按标题和内容搜索
- **多维度排序**：
  - ⏰ 时间排序（新-旧/旧-新）
  - 📏 字数统计（多-少/少-多）
  - 🔤 标题排序（A-Z/Z-A）

### ✅ **批量操作功能**
- 复选框多选对话
- 一键全选/取消全选
- 批量删除收藏
- 实时显示选中数量

### 🎨 **用户界面**
- 左右分栏布局（左侧分类，右侧内容）
- 自动适配系统深色/浅色主题
- 对话预览
- 分类下拉菜单快速标记

### 💾 **数据安全**
- 所有数据存储在浏览器本地
- 不会上传任何对话内容
- 防重复收藏检测（自动更新时间戳）

## 🚀 安装方法

### 前置条件
1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 复制[完整源代码](#)
3. 点击浏览器右上角Tampermonkey图标
4. 选择"添加新脚本"
5. 粘贴代码并保存（Ctrl+S）

## 📖 使用指南

### 收藏对话
1. 打开任意DeepSeek对话页面
2. 点击右下角 **⭐按钮** 即可收藏当前对话
3. 自动检测重复内容，重复收藏只更新时间戳

### 管理收藏
1. 点击右下角 **📚按钮** 打开管理器
2. **左侧分类栏**：点击切换分类视图
3. **搜索框**：输入关键词实时过滤
4. **排序下拉**：选择排序方式
5. **复选框**：勾选多个对话进行批量操作

### 分类管理
- **重命名分类**：鼠标悬停分类名 → 点击✎图标 → 输入新名称
- **移动对话**：在对话标题旁的下拉菜单中选择分类
- **分类统计**：每个分类后显示该分类收藏数量

## ⚙️ 配置说明

### 固定分类（可重命名）
```
1. 化学
2. 英语
3. 数学
4. 物理
5. 计算机
6. 学校琐事
```
*注：分类数量固定为6个，仅支持重命名，不支持增删*

### 存储说明
- 数据存储在浏览器的**GM_setValue**中
- 清除浏览器缓存会导致收藏丢失！
- 换设备无法同步

## 🔧 自定义修改

### 修改默认分类
在代码开头找到 `FIXED_CATEGORIES` 数组：
```javascript
const FIXED_CATEGORIES = [
    '化学',
    '英语',
    '数学',
    '物理',
    '计算机',
    '学校琐事'
];
```

## ❓ 常见问题

**Q：收藏的对话保存在哪里？** 
A：全部保存在浏览器本地，不会上传任何服务器。

**Q：如何备份收藏？** 
A：目前不支持直接导出。如需备份，可复制 Tampermonkey 存储的数据。

**Q：为什么按钮不显示？** 
A：确保在 `https://chat.deepseek.com/*` 域名下，刷新页面重试。

**Q：分类可以增加吗？** 
A：固定6个分类，但可以重命名为任何你想要的名字。

**Q：为什么有一个标题为“---”的对话，删不掉也修改不了？**

A：这是作者修不了的bug，因为作者不会写js。目前这是一个无伤大雅的b...，哦不是不是，这是一个好玩的feature...
